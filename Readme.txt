# Domino Effect
A highly optimized, vanilla-friendly chain destruction datapack for Minecraft 1.21+.

アイテムコンポーネントとマクロを活用し、1tickでの遅延のない一括破壊を実現しました。
独自のブロックタグ（`#peru:ores` など）を採用しているため、JSONファイルを1つ編集するだけで、Modで追加された独自の鉱石や原木にも簡単に対応させることができます。
